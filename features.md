## Home Page

## Dashboard

- Dashboard should have a section, with all live matches. Mobile view hall be a carousel of cards, desktop view will be a horizontally scrollable list of cards.
- Each card shall contain match title, match status, current score, and a button to view the scorecard.
- After live matches, we should have a similar section for live tournaments.
- After live tournaments, we should have a similar section for recently concluded matches.
- After recently concluded matches, we should have an action panel with the following actions:
  - Tournaments
  - Teams
  - Players
  - Create (This shall be a dropdown with the following options: Match, Tournament, Team, Player)

## Scorecard

- ✅ When user selects some run, dismissals except run out, if selected must be cleared.
- ✅ When user selects any dismissal except run out, runs if already added should be cleared.
- ✅ When user selects no ball, dismissals except run out must be cleared.
- ✅ When user selects wide ball, dismissals expect run out and stump out must be cleared.
