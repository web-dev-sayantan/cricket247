# Features

## Scorecard

- ✅ When user selects some run, dismissals except run out, if selected must be cleared.
- ✅ When user selects any dismissal except run out, runs if already added should be cleared.
- ✅ When user selects no ball, dismissals except run out must be cleared.
- ✅ When user selects wide ball, dismissals expect run out and stump out must be cleared.
